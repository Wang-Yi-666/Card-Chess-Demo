# report-20260331 四向走格移动配置说明

## 生效脚本
- `Scripts/Map/Actors/Player.cs`

## 当前行为
- 仅支持四向移动：上/下/左/右。
- 斜向输入会根据最近按键轴优先级转为单轴移动。
- 角色以“每次一格”的方式推进，不是自由连续漂移。

## 可调参数
1. `GridTileSize`
- 含义：每一步走多少像素（一个 tile 的尺寸）。
- 默认：16。
- 若你的地图 tile 不是 16，请改成对应值（如 24/32）。

2. `Speed`
- 含义：格子内移动速度（像素/秒）。
- 步时长近似为：`GridTileSize / Speed`。
- 示例：16 格 + 160 速度 = 0.1 秒/格。

3. `SnapToGridOnReady`
- 含义：进入场景时是否自动吸附到网格。
- 推荐：true（避免角色初始位置偏半格）。

## 推荐组合（复古手感）
- `GridTileSize = 16`
- `Speed = 120 ~ 180`
- `SnapToGridOnReady = true`

## 注意事项
- 若地图碰撞与 tile 不对齐，会出现“看起来能走但起步被挡”现象，需要把碰撞也按同网格对齐。
- 如果后续切到 `TileMapLayer` 且 tile size 改变，记得同步 `GridTileSize`。
