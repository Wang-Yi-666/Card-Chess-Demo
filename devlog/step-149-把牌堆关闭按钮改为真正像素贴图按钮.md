# step-149 把牌堆关闭按钮改为真正像素贴图按钮

时间：2026-04-05

本次改动目标：
- 修复牌堆右上角关闭按钮看起来发糊、不是真像素的问题。

本次实际改动：
- 修改 `Scripts/Battle/UI/BattleHudController.cs`
  - 删除关闭按钮原先基于字体和 `StyleBoxFlat` 的实现。
  - 改为程序生成的像素背景纹理 + 像素 `X` 图标纹理。
  - 按钮尺寸改为 `16x16`，图标尺寸为 `8x8`，全部按整像素居中摆放。
  - 关闭按钮根据 `normal / hover / pressed / disabled` 状态切换不同的像素底图与图标颜色。
  - 在 `Refresh()` 内增加关闭按钮布局刷新，避免尺寸或状态变化后出现偏移。

设计说明：
- 这次关闭按钮不再依赖字体渲染，因此不会再出现字体抗锯齿带来的模糊感。
- 背景和图标都使用 `Nearest` 过滤，并按固定像素尺寸直接绘制，保证显示结果是整像素。

验证方式：
1. 打开战斗牌堆弹窗。
2. 观察右上角 `X` 是否为方形像素按钮。
3. 鼠标悬停和按下时，底色是否有明显变化且无模糊。
