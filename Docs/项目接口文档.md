# 项目接口文档

## 1. 文档范围

本文件只列出当前代码里真实存在的接口、资源类型和关键类，并额外标注哪些类只是辅助/旧调试实现。

以下旧接口在当前仓库中不存在，因此本文件不再把它们写成现有接口：

- `BattleRequest`
- `BattleResult`
- `MapResumeContext`
- `TargetingService`
- `TurnFlowController`
- `EnergyState`
- `DeckState`

## 5. 房间与内容接口

### 5.1 `BattleRoomTemplate`

文件：

- `Scripts/Battle/Rooms/BattleRoomTemplate.cs`

当前职责：

- 维护 `FloorLayer` 和 `MarkerLayer`
- 在编辑器和运行时保证 TileSet 可用
- 从 marker layer 生成 `RoomLayoutDefinition`
- 提供屏幕坐标到格子的换算

当前关键接口：

```csharp
BoardTopology Topology { get; }
RoomLayoutDefinition BuildLayoutDefinition()
void SyncMarkersFromBoard(BoardObjectRegistry registry)
bool TryScreenToCell(Vector2 globalPosition, out Vector2I cell)
Vector2 CellToLocalCenter(Vector2I cell)
Rect2 GetCellRect(Vector2I cell)
bool SupportsEnemyTypes(IReadOnlyList<string> enemyTypeIds, out bool exactMatch)
```

### 5.2 `BattleRoomPoolDefinition` / `BattleRoomPoolEntry`

文件：

- `Scripts/Battle/Rooms/BattleRoomPoolDefinition.cs`
- `Scripts/Battle/Rooms/BattleRoomPoolEntry.cs`

当前用途：

- 为 `BattleSceneController` 提供房间池

当前字段：

- `BattleRoomPoolDefinition.Entries`
- `BattleRoomPoolEntry.EnemyTypeId`

### 5.3 `BattleRoomTileSetFactory`

- `Scripts/Battle/Rooms/BattleRoomTileSetFactory.cs`

### 6.6 当前默认 prefab 场景

当前默认资源绑定为：

- `battle_player -> Scene/Battle/Tiles/BattlePlayerToken.tscn -> BattlePlayerView`
- `battle_enemy -> Scene/Battle/Tiles/BattleEnemyToken.tscn -> BattleEnemyView`
- `battle_obstacle -> Scene/Battle/Tiles/BattleObstacleToken.tscn -> BattleObstacleView`

说明：

- `DefaultBattlePrefabLibrary.tres` 当前引用的是 `Scene/Battle/Tiles/*.tscn`
- 这些 token 场景挂接的是 `Presentation` 下的 `Battle*View`，不是 `Actors` 目录下那组脚本

## 8. 场景控制接口

### 8.1 `BattleSceneController`

文件：

- `Scripts/Battle/BattleSceneController.cs`

当前导出配置：

- `ForcedBattleRoomScene`
- `BattleRoomScenes`
- `BattleRoomPools`
- `BattlePrefabLibrary`
- `EncounterEnemyTypeIds`
- `RandomSeed`
- `CameraZoom`
- `CameraTopMarginPixels`

当前只读状态：

- `BoardState`
- `Registry`
- `QueryService`
- `TurnState`
- `CurrentRoom`
- `GlobalSession`
- `StateManager`

当前职责：

- 初始化棋盘和房间
- 创建 registry / query / turn state
- 创建表现层和 HUD
- 接收左键移动、回合切换、移动力调试输入

当前不负责的内容：

- 地图切换
- 战斗结果输出
- 卡牌解析
- 普攻 / 防御 / 冥想
- 时间推进
